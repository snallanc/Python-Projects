KEY RECURRENCES:
  Fibonacci / Climbing Stairs : dp[i] = dp[i-1] + dp[i-2]
  House Robber                : dp[i] = max(dp[i-1], dp[i-2] + nums[i])
  Two Sum (unsorted)          : for n at i, check if (target-n) in seen; else seen[n] = i
  Two Sum (sorted)            : l++  if sum < target | r-- if sum > target
  3Sum                        : sort + fix nums[i] + two pointers on rest | nums[i]>0 → break
                              : skip dup i (i>0 and nums[i]==nums[i-1]) | skip dup p0/p1 after match
  Container With Most Water   : area = min(h[l], h[r]) * (r-l); move pointer at shorter line
  Trapping Rain Water         : move side with smaller height; add bounded water using left_max/right_max
  Longest Unique Substr       : if c in seen and seen[c] >= l: l = seen[c]+1 | seen[c] = r
  Daily Temperatures          : monotonic decreasing stack of unresolved indices; pop on warmer day
  Validate Brackets           : push opens; on close, top must be matching opener; stack empty at end
  Subarray Sum Multiple of k  : if rem seen before and idx-prev_idx >= 2 => valid subarray
                              : rem = prefix_sum % k (or prefix_sum when k == 0)
  Remove Adjacent Dups (k)    : push/increment (char,count); pop when count == k
  Reverse Endian (32-bit)     : swap bytes with masks/shifts: b0<->b3, b1<->b2
  Product Except Self         : pass1 (right-to-left): res[i] = suffix product before i
                              : pass2 (left-to-right): res[i] *= prefix product before i
  Max Stock Profit            : optimized: sum of positive day-to-day gains
                              : profit += max(0, prices[i] - prices[i-1])

CRITICAL METHODS:
  Heap (heapq)    : heapq.heappush(h, x)          → push, maintains heap property
                  : heapq.heappop(h)              → pop smallest; negate values for max-heap
                  : heapq.heapify(list)           → in-place O(n) build
                  : heapq.nsmallest/nlargest(k,h) → top-K without full sort
  LRU/MRU Cache  : od.move_to_end(key, last=True) → move to back (MRU side)
                  : od.move_to_end(key, last=False)→ move to front (LRU side)
                  : od.popitem(last=False)         → evict LRU (front)
                  : od.popitem(last=True)          → evict MRU (back)
  Union-Find     : find(x) with path compression  → parent[x] = find(parent[x])
                  : union(a,b) with rank           → attach smaller tree under larger root
  Graph          : graph[v].append((neighbor, wt)) → weighted adjacency list
                  : for n, _ in graph[v]           → unpack (neighbor, weight) tuples

Container        | Lookup | Insert | Interview Use Case
-----------------|--------|--------|----------------------------------------------------
dict             | O(1)   | O(1)   | Two-sum, memoization, frequency count
set              | O(1)   | O(1)   | Duplicate check, visited tracking, existence
list             | O(1)*  | O(1)** | Sliding window, two pointers, DP table
Counter          | O(1)   | O(1)   | Top-K frequent, anagram check, window frequency
defaultdict(list)| O(1)   | O(1)   | Group anagrams, build adjacency list
deque            | O(n)   | O(1)†  | BFS queue, sliding window max
OrderedDict      | O(1)   | O(1)   | LRU/MRU cache (move_to_end + popitem)

* O(1) by index, O(n) by value search
** O(1) amortized append; O(n) insert at arbitrary position
† O(1) append/pop from both ends

CRITICAL METHODS:
  dict            : d.get(k, default)             → safe lookup without KeyError
                  : d.setdefault(k, []).append(v) → group items without explicit init check
                  : d.items()                     → iterate (key, value) pairs
  set             : s.add(x) / s.discard(x)       → add or safely remove
                  : a & b, a | b, a - b           → intersection, union, difference (O(min))
  Counter         : c.most_common(k)              → top-K in O(n log k)
                  : Counter(a) & Counter(b)        → min-count intersection (anagram check)
                  : c[key] += 1 / -= 1            → sliding window frequency update
  defaultdict     : d[k].append(v)               → no KeyError; auto-init on first access
  deque           : dq.appendleft(x)/popleft()   → O(1) both ends; use for BFS queue
                  : dq.append(x)/pop()            → right-end ops; max-window monotonic deque
  OrderedDict     : od.move_to_end(k, last=True)  → mark recently used (LRU/MRU)
                  : od.popitem(last=False/True)    → evict LRU (False) or MRU (True)
