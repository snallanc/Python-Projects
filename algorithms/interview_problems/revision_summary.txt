File                            | Problem                        | Pattern               | Data Structure      | Time    | Space
--------------------------------|--------------------------------|-----------------------|---------------------|---------|------
fibonacci.py                    | Nth Fibonacci number           | 1D DP (space opt.)    | Two variables       | O(n)    | O(1)
climbing_steps.py               | Ways to climb n stairs (1/2)   | 1D DP (= Fibonacci)   | Two variables       | O(n)    | O(1)
house_robber.py                 | Max sum non-adjacent elements  | 1D DP                 | Two variables       | O(n)    | O(1)
find_duplicates.py              | Any duplicates in list?        | Set membership        | set                 | O(n)    | O(n)
longest_consequtive_sequence.py | Longest consecutive sequence   | Set + greedy          | set                 | O(n)    | O(n)
two_sum_sorted.py               | Two numbers summing to target  | Two Pointers          | Two indices         | O(n)    | O(1)
union_find.py                   | Connected components / cycles  | Union-Find (DSU)      | parent[], rank[]    | O(α(n)) | O(n)
LRU_MRU_cache.py                | LRU / MRU cache with eviction  | OrderedDict hash map  | OrderedDict         | O(1)    | O(cap)

KEY RECURRENCES:
  Fibonacci / Climbing Stairs : dp[i] = dp[i-1] + dp[i-2]
  House Robber                : dp[i] = max(dp[i-1], dp[i-2] + nums[i])
  Two Sum (sorted)            : l++  if sum < target | r-- if sum > target

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Need O(1) lookup?                 → dict / set
  Need to count things?             → Counter
  Need to group things?             → defaultdict(list)
  Need fast both-end ops?           → deque
  Need min/max repeatedly?          → heap (MinHeap/MaxHeap)
  Need order + O(1) move/evict?     → OrderedDict
  Sorted array + find pair?         → Two Pointers
  Subarray/substring condition?     → Sliding Window
  "How many ways..." ?              → Counting DP
  "Max/min value..." ?              → Optimization DP
  Shortest path (unweighted)?       → BFS
  Shortest path (weighted)?         → Dijkstra's
  Connected components / cycles?    → Union-Find or DFS
  Prefix problems on strings?       → Trie
  Ordered insertions + BST?         → BST
